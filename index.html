Software

  1. lwdid  |  GitHub repository

    lwdid is a Stata package implementing rolling difference-in-differences estimators for panel data, suitable for both small-N and large-N settings.

    Installation

    ssc install lwdid, replace

    Stata Module Citation

    Lee, S. J. and Wooldridge, J. M. (2026). lwdid: Stata module to implement rolling difference-in-differences estimator for small-N and large-N panel data. Statistical Software Components S459672, Boston College Department of Economics, revised 1 June 2026.

    For detailed syntax, options, and examples, please see Rolling Difference-in-Differences Estimation for Small and Large Panels (with Elizabeth Kayoon Hur and Jeffrey M. Wooldridge).
